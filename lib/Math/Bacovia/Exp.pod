=encoding utf8

=head1 NAME

Math::Bacovia::Exp - Represents the natural exponentiation of a symbolic value.

=head1 SYNOPSIS

    use 5.014;
    use Math::Bacovia qw(Exp Symbol);

    my $f = Exp(Symbol('a'));
    my $g = Exp(Symbol('b'));

    say (($f * $g)->pretty);    #=> exp(a + b)

=head1 METHODS

This section describes the methods provided by the B<Math::Bacovia::Exp> module.

=head2 new

    my $obj = Math::Bacovia::Exp->new($value);

Constructs and returns a new B<Math::Bacovia::Exp> object.

=head2 get

    my ($value) = $obj->get;

Returns the internal value of the self-object.

=head2 mul

    $x * $y
    $x->mul($y)

Product of C<x> and C<y>, using the identity:

    exp(a) * exp(b) = exp(a + b)

=head2 div

    $x / $y
    $x->div($y)

Division of C<x> by C<x>, using the identity:

    exp(a) / exp(b) = exp(a - b)

=head2 inv

    -$x
    $x->inv

Multiplicative inverse of C<x>, using the identity:

    1/exp(a) = exp(-a)

=head2 eq

    $x == $y
    $x->eq($y)

Return a true value when C<x> and C<y> are symbolically equal.

=head1 SPECIAL METHODS

This section describes the special methods provided by the B<Math::Bacovia::Exp> module.

=head2 alternatives

    my @alt = $obj->alternatives;
    my @alt = $obj->alternatives(full => 1);

This method uses common mathematical identities to create symbolically equivalent expressions from the self-expression.

=head2 pretty

    my $str = $obj->pretty;

Returns a human-readable stringification of the self-expression.

=head2 stringify

    my $str = $obj->stringify;

Returns a stringification of the self-expression.

=head2 numeric

    my $num = $obj->numeric;

Evaluates the self-expression numerically and returns a L<Math::AnyNum> object.
